# Real-Time Transport Tracking and Smart Bus Monitoring System

## 📌 Project Overview

The Real-Time Transport Tracking and Smart Bus Monitoring System is a mobile-based application designed to provide live bus tracking and transport monitoring using GPS technology. The system enables passengers to view the real-time location of buses, monitor routes, and receive accurate transport information through an intuitive mobile application.

The application is developed using Flutter for the frontend, Firebase for backend services and real-time data synchronization, and Google Maps API for live location visualization. GPS data collected from the driver's mobile device is continuously updated to the cloud, allowing passengers and administrators to monitor vehicle movement in real time.

This project aims to improve transportation efficiency, reduce passenger waiting time, and enhance the overall travel experience through smart transport management.

## ⚙️ System Workflow

1. Driver starts the trip using the mobile application
2. GPS continuously captures live vehicle coordinates
3. Location data is sent to Firebase Realtime Database
4. Passenger application retrieves real-time coordinates
5. Google Maps displays the live bus location and route information
6. Users can monitor vehicle movement and transport status in real time
